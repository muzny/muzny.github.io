This page contains the data and code associated with the paper (please cite this if you use this data):

Felix Muzny, Mark Algee-Hewitt and Dan Jurafsky. 2017. Dialogism in the novel: A computational model of the dialogic nature of narration and quotations. Digital Scholarship in the Humanities. bib
Digital Humanities NLP

Data

We release the metadata for the corpus associated with the journal article "Dialogism in the novel: A computational model of the dialogic nature of narration and quotations.

Corpus Metadata

A description of each field in the data is listed below.

  • author: the author of the text
  • title: the title of the text
  • date: the publication date of the text
  • type: whether this row corresponds to the extracted narration or dialog
  • dialogism_score: the dialogism score assigned to the narration or dialog in this text
  • num_spans: the number of continguous spans of text of narration or dialog that were extracted
  • num_words: the number of words of narration or dialog that were extracted

We further release the hand-annotated data for evaluation of the performance of our quote extraction method. Note that this is annotated for quote boundaries and gender of the speaker but not specific character. Data found in the QuoteLi page is annotated for both quote boundaries and specific character speakers.
